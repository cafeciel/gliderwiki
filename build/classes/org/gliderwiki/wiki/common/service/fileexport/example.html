Ubuntu에서는 /etc/apache2/conf.d/security 파일에 설정한다.

불필요한 정보 숨기기

  • ServerTokens Prod : HTTP 헤더에 불필요한 서버 관련 상세정보 제거
  • ServerSignature Off : Apache가 자동으로 보여주는 페이지들에서 서버관련 상세정보 제거
  • TraceEnable Off : TRACE 메소드 거부
