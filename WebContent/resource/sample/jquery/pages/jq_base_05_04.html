[toggle:선택한 엘리먼트에 두 개의 이벤트 핸들러를 연결]

AAAAA BBBBB 
 jQuery(document).ready(function(){
	$(".buttons").toggle(
		function(){
		$(this).addClass("hover");
		},
		function(){
		$(this).removeClass("hover");
		}
	);
 });

첫 번째 이벤트 핸들러 함수는 0부터 시작한다고 할 경우 이벤트 짝수 번째로 발생한 경우에 실행되고, 두번째 이벤트 핸들러 함수는 이벤트가 홀수 번째로 발생한 경우에 실행
이벤트가 처음 발생하면 첫 번째 이벤트 핸들러 함수가 실행되고 다시 발생하면 두번째 이벤트 핸들러 함수가 실행
이벤트 발생 시마다 이벤트 핸들러 함수가 순환
